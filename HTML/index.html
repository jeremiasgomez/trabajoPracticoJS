  5. Los operadores y y conversión entre tipos son:
    Lógicos
    Realiza aplicaciones complejas, ya que se utilizan para tomar decisiones sobre las instrucciones que debería ejecutar el programa en función de ciertas condiciones
    Negación
    Es uno de los mas utilizados. Se utiliza para obtener el valor contrario al valor de la variable:
    Incremento y decremento
    Estos dos operadores solamente son válidos para las variables numéricas y se utilizan para incrementar o decrementar en una unidad el valor de una variable
    Asignación
    Este operador se utiliza para guardar un valor específico en una variable.
  6. Las estructuras de control en JavaScript son las siguientes:
    For= Un bucle for se repite hasta que la condición que se esta evaluando se cumpla.
    While= Ejecuta una sentencia mientras la condición que se este evaluando sea verdadera.
    Condiciones= Con las condicionales podemos evaluar si una condición cumple los requisitos con lo que estemos evaluando.
    Switch= Permite evaluar una expresión e intenta igual el valor de esa expresión a una etiqueta llamada case y si en el caso de que la condición se cumpla o lo que tiene el case, se ejecuta la sentencia.
  7. El objeto dom es es una interfaz de programación para los documentos, el cual trabaja en un formato de documento para que el html pueda ser leido y entendido por el documento JavaScript desde las estiquetas mas grandes hasta las mas chicas.
  8. El object map() en JavaScript funciona llamando a todos los objetos que hay en una variable hasta otra variable mencionando todo lo que hay de pormedio usualmente usandose de una manera en que una array pueda convertirse en parte de otro array
